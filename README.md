# rock-paper-scissors-python

A simple Python-based Rock Paper Scissors game where a user plays against the computer.

## Features

* Random computer choice
* User score tracking
* Computer score tracking
* Input validation
* Round counter
* First to 3 points wins

## Technologies Used

* Python
* Random Module

## How to Run

1. Download the project.
2. Open a terminal.
3. Run:

python rock_paper_scissors.py
